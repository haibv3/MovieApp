# MovieApp

A modern Android movie app built with the latest Android development tools and best practices.

## Features
- Browse popular movies
- Search movies
- View movie details
- Save favorite movies
- Offline support

## Tech Stack
- **UI**: Jetpack Compose
- **Architecture**: Clean Architecture + MVVM
- **DI**: Hilt
- **Database**: Room
- **Network**: Retrofit
- **Async**: Kotlin Coroutines + Flow
- **Image Loading**: Coil
